# Roshni Hospital Management System

This is a simple Java-based Hospital Management System project demonstrating:

- Abstraction
- Inheritance
- Interfaces
- Polymorphism
- Object-Oriented Programming concepts

## Classes Included

- Person (Abstract Class)
- Staff (Abstract Class)
- Patient
- Doctor
- Billing
- Billable (Interface)
- Schedulable (Interface)

This project was created using Java in Visual Studio Code.
